0. 选择困难症

前端三大框架, 各有优劣, 网上各种测评看了好久, 最终决定先拿 vue 下手
然后问题又来了, 用什么 UI 框架呢?

  • vux
  • element-ui
  • mint-ui
  • bootstrap-vue
  • ……

最终选择了星星数最多的 vux, 虽然这个框架是个人开发维护的, 也存在很多问题.

1. vue-cli3 与 vux 的融合

vux 官方暂时还没有适配 vue-cli3
vue-cli3 可以用 vue-cli-plugin-vux 一键安装 vux UI 框架, 不需要手动适配 vue-loader

2. 初始化项目的注意事项

可以通过命令行输入vue ui打开图形化界面
也可以vue create `project-name`直接在命令行创建
两种方式的效果是一样的, 看你习惯用哪种
